Privacy Policy

Last updated: 2026-01-14

This application is a personal automation tool that connects to my Google account (Gmail and Google Calendar) to run workflows locally using n8n.

Data processed

The app may access and process Gmail and Google Calendar data depending on the permissions granted (e.g., email content and metadata, calendar events). Data is used only to execute the configured automations.

Data storage

No Gmail or Google Calendar data is stored permanently. Data is processed locally for automation purposes and then discarded. The application does not maintain its own copies of emails, events, or attachments.

Credentials and tokens

OAuth credentials/tokens required to access Google APIs are stored locally by my self-hosted n8n instance running inside a Docker container, using n8n’s credential storage/encryption mechanisms.

Data sharing

No data is shared with third parties. All processing happens locally on my machine.

Security

Access is restricted to my local environment. No external users have access to the application or its data.

User controls and revocation

You can revoke the app’s access at any time in your Google Account settings under Security → Third-party access (or “Manage third-party access”). After revocation, the app will no longer be able to access Gmail/Calendar.

Contact

For questions about this policy, contact: iraitz.cordero@gmail.com

This policy is intended for personal use of a self-hosted automation setup and applies only to this application.
