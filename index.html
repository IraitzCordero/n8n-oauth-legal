Automatizaciones n8n

Personal self-hosted automations (n8n running locally in Docker).

This application is used exclusively to run personal automation workflows using n8n (self-hosted). It connects to Google services to execute user-configured automations.

The n8n web interface is private and protected by authentication (it is not a public website).

Google services used

  • Gmail: read/search emails, apply labels, move messages, create drafts, send emails
  • Google Calendar: create, read and update events
  • Google Drive: manage folders/files and Google Workspace documents (Docs, Sheets, etc.)
  • Google Analytics: read-only access for reporting and analysis
