Automatizaciones n8n

Automatizaciones personales autoalojadas (n8n en Docker).

Esta aplicación se usa únicamente para ejecutar automatizaciones personales que se conectan a mi cuenta de Google para procesar datos de Gmail y Google Calendar.

Todo el procesamiento se realiza localmente en una instancia privada de n8n. No se almacena de forma permanente el contenido de correos ni eventos.

La interfaz de n8n está protegida con autenticación (no es pública).
